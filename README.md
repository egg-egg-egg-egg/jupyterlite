# 介绍

这是一个运行在网页的python，也就是说你所有的python代码都会在你的网页中运行，所有的文件也都会保存到你的本地。

无论是手机、平板、电脑只要有这个链接都可以使用，无需安装任何额外的内容。

- 目前暂时不能使用海龟库。
